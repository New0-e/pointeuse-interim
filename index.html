Configuration Google
Saisissez votre Client ID OAuth 2.0 pour activer la synchronisation Drive.
📖 Comment obtenir votre Client ID :

1. console.cloud.google.com → créer/sélectionner un projet
2. APIs & Services → OAuth consent screen
  → Type : External → ajoutez votre email en "Test user"
3. Credentials → + Create Credentials → OAuth client ID
4. Type : Web application
5. Authorized JavaScript origins : URL de cette page
  (ex : http://localhost:3000 ou votre domaine)
6. Copiez le Client ID ci-dessus

⚠️ Le secret client n'est jamais nécessaire ici.
Ne le partagez et ne l'intégrez jamais dans le code.
🔐
Connexion à Google Drive
Vos données seront stockées dans un fichier JSON sur votre Drive personnel. Aucune donnée ne transite par un serveur tiers.
